# my-app
Vue2开发的通用后台管理系统，实现了登录认证、菜单权限控制、数据可视化、用户管理等功能。

采用前后端分离架构，通过Mock.js 模拟前端接口完成开发。

Vue3重构项目地址https://github.com/ting131419/vue3-admin

## Project setup
```
yarn install
```
